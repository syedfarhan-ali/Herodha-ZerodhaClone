## 🌐 Herodha – Stock Dashboard Web App  
**Tech Stack:** MongoDB | Express.js | React | Node.js | Chart.js  

A modern stock trading dashboard inspired by Zerodha that allows users to track their stock portfolio, analyze performance, and visualize market trends using interactive charts.

---

### 🚀 Key Highlights

- 📊 Interactive charts using Chart.js  
- 💼 Portfolio dashboard to display stock positions  
- 📈 Clean UI inspired by Zerodha  
- 🔄 Dynamic stock data rendering  
- ⚛️ Component-based architecture with React  
- ⚡ Fast and responsive design  

---

### 🛠️ Features

- 📃 View stock holdings with:
  - Quantity  
  - Average price  
  - Current price  
  - Profit/Loss  
- 📉 Graphical representation of stock performance  
- 🔍 Minimal and intuitive dashboard UI  
- 📱 Fully responsive layout
